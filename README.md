# Super Remove 

Plugin help us remove file and floder in command line


## Supported OperaSystem
* Window
* MacOSX
* Linux

## Install NodeJS
[Install NodeJs click here !](https://nodejs.org/en/)

## Install

```javascript npm i -g vnrm```
